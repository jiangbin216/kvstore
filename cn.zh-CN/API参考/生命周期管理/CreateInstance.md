# CreateInstance {#doc_api_R-kvstore_CreateInstance .reference}

调用CreateInstance创建一个Redis实例。

该API对应的控制台操作请参见[创建实例](~~26351~~)。

创建Redis实例所需的实例规格请参见[实例规格表](~~107984~~)。
